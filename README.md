# ChatbotNLP 
A question-answer chatbot NLP using 5 different models to classify user intents and provide responses in Arabic and English.  The API is deployed with FastAPI and hosted on Hugging Face Spaces.

# Models Used
- BERT
- SVM
- GRU
- RAG
- LSTM
  
# Deployment
Hosted links for best models (SVM, RAG) for results :- 
RAG Model:
https://huggingface.co/spaces/tasabehahmed/Chatbot-Bert
SVM Model:
https://huggingface.co/spaces/tasabehahmed/ChatbotNLP-SVM

# Tech Stack
- Python
- FastAPI
- scikit-learn
- PyTorch
- Hugging Face Transformers
- NLP preprocessing (tokenization, vectorization)

  # Objective
1- Build a domain-specific NLP chatbot specialized in answering user questions about the best places to buy electrical and home appliances.
2-Deploy a scalable chatbot API that can be easily integrated into web applications.
